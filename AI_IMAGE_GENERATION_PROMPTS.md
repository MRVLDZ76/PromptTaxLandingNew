## Social Media Image Dimensions Reference

### **Facebook/Open Graph (OG)**
- **Dimensions**: 1200x630 pixels (1.91:1 ratio)
- **Format**: JPEG or PNG
- **Max file size**: 8MB
- **Usage**: Facebook, LinkedIn (compatible), general OG tags

### **LinkedIn**
- **Dimensions**: 1200x627 pixels (1.91:1 ratio)
- **Format**: JPEG or PNG
- **Max file size**: 5MB
- **Usage**: LinkedIn articles and posts (slightly taller than Facebook)

### **Twitter**
- **Dimensions**: 1200x675 pixels (16:9 ratio)
- **Format**: JPEG or PNG
- **Max file size**: 5MB
- **Card Type**: summary_large_image
- **Usage**: Twitter cards, wide-format social sharing

### **Instagram (Optional)**
- **Dimensions**: 1080x1080 pixels (1:1 ratio)
- **Format**: JPEG or PNG

## General Best Practices for AI Image Generation

### For Leonardo.ai / Midjourney:
1. **Aspect Ratio Flags**:
   - Facebook/OG: `--ar 1.91:1` or `--ar 19:10`
   - LinkedIn: `--ar 1.91:1` or `--ar 1200:627`
   - Twitter: `--ar 16:9`

2. **Style Modifiers**:
   - Add: `professional illustration style`, `clean vector art`, `corporate design`
   - For photos: `photo-realistic`, `professional photography`, `studio lighting`
   - For tech: `modern UI design`, `SaaS dashboard aesthetic`, `fintech style`

3. **Quality Settings**:
   - Use `--quality 2` or `--q 2` for higher quality
   - Add `--stylize 50` for moderate stylization (not too artistic)

### For DALL-E 3:
1. **Size Specifications**:
   - Use: `1792x1024` (landscape) - closest to required ratios
   - Then resize in Photoshop/Figma to exact dimensions

2. **Prompt Structure**:
   - Be specific about composition: "horizontal composition", "landscape orientation"
   - Specify text space: "leave space at top for headline text"
   - Request: "professional business illustration", "corporate aesthetic"

### Post-Processing Steps:
1. **Resize to exact dimensions** in Photoshop/Canva:
   - 1200x630 (Facebook/OG)
   - 1200x627 (LinkedIn) 
   - 1200x675 (Twitter)

2. **Optimize for web**:
   - Save as JPEG at 80-85% quality
   - Keep file size under 2MB for faster loading
   - Use tools like TinyPNG or ImageOptim

3. **Add text overlays** (optional):
   - Use Canva or Figma to add blog title text
   - Keep text readable on mobile (minimum 24px font size)
   - Ensure high contrast between text and background

4. **Test across platforms**:
   - Use Facebook Sharing Debugger
   - LinkedIn Post Inspector
   - Twitter Card Validator

## Quick Command Summary for AI Tools

### Midjourney Format:
```
/imagine [your prompt] --ar 1.91:1 --q 2 --s 50
```

### Leonardo.ai Format:
- Select "Alchemy" for better quality
- Choose "Photo Real" or "Illustration" model
- Set dimensions manually in advanced settings
- Use negative prompts: "blurry, low quality, watermark, text"

### DALL-E 3 Format:
```
Generate: [your prompt]
Size: 1792x1024 (landscape)
Quality: HD
Style: natural (for photos) or vivid (for illustrations)
```
